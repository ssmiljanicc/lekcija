### 1. Use `removeprefix` instead of `lstrip` for current-branch marker stripping
**File**: `branch_counter.py:43`
**Type**: Improved clarity / Consistency
**Impact**: Low

**Before:**
```python
marker = "* " if b.startswith("*") else "  "
print(f"  {marker}{b.lstrip('* ')}")
```

**After (suggested):**
```python
marker = "* " if b.startswith("*") else "  "
print(f"  {marker}{b.removeprefix('* ')}")
```

**Why:** `str.lstrip('* ')` strips individual characters from the set `{'*', ' '}` from the left — not the literal prefix `"* "`. For all realistic git branch names the result is identical, but `removeprefix` is semantically precise (strips the exact prefix once), consistent with line 47 which already uses `removeprefix('remotes/')`, and avoids a subtle class of bugs if a branch name ever starts with characters in the strip set.
**Functionality:** Preserved (identical behavior for all git branch names)
